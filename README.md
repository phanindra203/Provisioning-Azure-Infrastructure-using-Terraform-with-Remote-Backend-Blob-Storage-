# Provisioning-Azure-Infrastructure-using-Terraform-with-Remote-Backend-Blob-Storage-
Provisioned cloud infrastructure on Microsoft Azure using Terraform, deploying VM and networking resources. Configured a remote backend with Azure Blob Storage for centralized state management, ensuring consistent, scalable, and repeatable Infrastructure as Code deployments.


# 🚀 Azure Infrastructure Deployment using Terraform

## 📌 Project Overview
This project demonstrates how to provision infrastructure in Microsoft Azure using Terraform and configure a remote backend using Azure Blob Storage.

## 🧱 Infrastructure Created
- Resource Group
- Virtual Network (VNet)
- Subnet
- Public IP
- Network Interface
- Linux Virtual Machine

## ⚙️ Tools & Technologies
- Terraform
- Microsoft Azure
- Azure Blob Storage (Remote Backend)

## 🔐 Key Features
- Remote state management using Azure Blob Storage
- Infrastructure as Code (IaC)
- Centralized state storage
- Scalable and repeatable deployments
